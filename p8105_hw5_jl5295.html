The code chunk below loads the iris dataset from the tidyverse package and introduces some missing values in each column. The purpose of this problem is to fill in those missing values. There are two cases to address:

For numeric variables, you should fill in missing values with the mean of non-missing values. For character variables, you should fill in missing values with virginica

Write a function that takes a vector as an argument; replaces missing values using the rules defined above; and returns the resulting vector. Apply this function to the columns of iris_with_missing using a map statement.

set.seed(10)

iris_with_missing = iris %>% 
  map_df(~replace(.x, sample(1:150, 20), NA)) %>%
  mutate(Species = as.character(Species))
fill_missing_values = function(x) {
  if (is.numeric(x)) {
  replace(x, is.na(x), mean(x, na.rm = TRUE)) 
}
  
  else if (!is.numeric(x)) {
    replace(x, is.na(x), "virginica")
  }
}

iris_without_missing = map_df(iris_with_missing, ~fill_missing_values(.x))

Comment: A new dataframe iris_without_missing was generated to test if the function works. iris_without_missing has correctly computed the mean value for all numeric missing values and has applied to word virginica to all missing character values.
